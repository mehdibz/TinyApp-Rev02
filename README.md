# Tinyapp Project

TinyApp is a full stack web appliction built with Node and Express that allows users to shorten long URLs (like bit.ly).

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

### Usage

Install the dependencies and start the server.

```
npm install
node express.js
open http://localhost:8080/
```
